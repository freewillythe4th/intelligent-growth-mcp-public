# Contributing

This repository accepts improvements to public setup guidance, examples and troubleshooting.

Before opening a pull request:

1. Keep examples focused on a real product marketing job.
2. Do not include credentials, customer records or private company content.
3. Do not add server implementation, protected method definitions or deployment configuration.
4. Keep client instructions accurate and conditional on plan availability.
5. Run `./scripts/audit-public.sh`.

For product method requests, open a feature request and describe the job, context and useful output. Do not propose an exported prompt pack.
